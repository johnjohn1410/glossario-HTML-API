Tags HTML

Tag Descrição
<!--...--> Define um comentário
<!DOCTYPE>  Define o tipo de documento
<a> Define um hyperlink
<abbr> Define uma abreviação ou um acrônimo
<address> Define as informações de contato do autor/proprietário de um documento
<area> Define uma área dentro de um mapa de imagem
<article> Define um artigo
<aside> Define conteúdo à parte do conteúdo principal da página
<audio> Define conteúdo de áudio embutido
<b> Define texto em negrito
<base> Especifica a URL/base para todos os URLs relativos no documento
<bdi> Isola uma parte de texto que pode ser formatada em uma direção diferente do texto ao seu redor
<bdo> Substitui a direção de texto atual
<blockquote> Define uma seção citada de outra fonte
<body> Define o corpo do documento
<br> Define uma quebra de linha simples
<button> Define um botão clicável
<canvas> Usado para desenhar gráficos, em tempo real, via scripts (geralmente JavaScript)
<caption> Define a legenda de uma tabela
<cite> Define o título de uma obra
<code> Define um trecho de código de computador
<col> Especifica as propriedades das colunas dentro de um elemento <colgroup>
<colgroup> Especifica um grupo de uma ou mais colunas em uma tabela para formatação
<data> Adiciona uma tradução legível por máquina de um determinado conteúdo
<datalist> Especifica uma lista de opções predefinidas para controles de entrada
<dd> Define uma descrição/valor de um termo em uma lista de descrição
<del> Define texto que foi deletado de um documento
<details> Define detalhes adicionais que o usuário pode ver ou ocultar
<dfn> Especifica um termo que será definido dentro do conteúdo
<dialog> Define uma caixa de diálogo ou janela
<div> Define uma seção em um documento
<dl> Define uma lista de descrições
<dt> Define um termo/nome em uma lista de descrições
<em> Define texto enfatizado
<embed> Define um contêiner para uma aplicação externa
<fieldset> Agrupa elementos relacionados em um formulário
<figcaption> Define uma legenda para um elemento <figure>
<figure> Especifica conteúdo auto-contido
<footer> Define um rodapé para um documento ou seção
<form> Define um formulário HTML para entrada de usuário
<h1> a <h6> Define títulos HTML
<head> Contém metadados/informações para o documento
<header> Define um cabeçalho para um documento ou seção
<hgroup> Define um cabeçalho e conteúdo relacionado
<hr> Define uma mudança temática no conteúdo
<html> Define a raiz de um documento HTML
<i> Define uma parte de texto em voz ou humor alternativo
<iframe> Define um quadro embutido (inline)
<img> Define uma imagem
<input> Define um controle de entrada
<ins> Define texto que foi inserido em um documento
<kbd> Define a entrada do teclado
<label> Define um rótulo para um elemento <input>
<legend> Define uma legenda para um elemento <fieldset>
<li> Define um item de lista
<link> Define o relacionamento entre um documento e um recurso externo (mais utilizado para vincular folhas de estilo)
<main> Especifica o conteúdo principal de um documento
<map> Define um mapa de imagem
<mark> Define texto destacado
<menu> Define uma lista não ordenada
<meta> Define metadados sobre um documento HTML
<meter> Define uma medição escalar dentro de um intervalo conhecido (um medidor)
<nav> Define links de navegação
<noscript> Define conteúdo alternativo para usuários que não suportam scripts do lado do cliente
<object> Define um contêiner para uma aplicação externa
<ol> Define uma lista ordenada
<optgroup> Define um grupo de opções relacionadas em uma lista suspensa
<option> Define uma opção em uma lista suspensa
<output> Define o resultado de um cálculo
<p> Define um parágrafo
<param> Define um parâmetro para um objeto
<picture> Define um contêiner para múltiplos recursos de imagem
<pre> Define texto pré-formatado
<progress> Representa o progresso de uma tarefa
<q> Define uma citação curta
<rp> Define o que mostrar em navegadores que não suportam anotações Ruby
<rt> Define uma explicação/pronúncia de caracteres (para tipografia asiática)
<ruby> Define uma anotação ruby (para tipografia asiática)
<s> Define texto que não está mais correto
<samp> Define a saída de exemplo de um programa de computador
<script> Define um script do lado do cliente
<search> Define uma seção de pesquisa
<section> Define uma seção em um documento
<select> Define uma lista suspensa
<small> Define texto menor
<source> Define múltiplos recursos de mídia para elementos de mídia (<video> e <audio>)
<span> Define uma seção em um documento
<strong> Define texto importante
<style> Define informações de estilo para um documento
<sub> Define texto subscrito
<summary> Define um título visível para um elemento <details>
<sup> Define texto sobrescrito
<svg> Define um contêiner para gráficos SVG
<table> Define uma tabela
<tbody> Agrupa o conteúdo do corpo de uma tabela
<td> Define uma célula em uma tabela
<template> Define um contêiner para conteúdo que deve ser ocultado quando a página é carregada
<textarea> Define um controle de entrada multilinha (área de texto)
<tfoot> Agrupa o conteúdo do rodapé em uma tabela
<th> Define uma célula de cabeçalho em uma tabela
<thead> Agrupa o conteúdo do cabeçalho em uma tabela
<time> Define um tempo específico (ou data/hora)
<title> Define o título do documento
<tr> Define uma linha em uma tabela
<track> Define faixas de texto para elementos de mídia (<video> e <audio>)
<u> Define texto não articulado e estilizado de maneira diferente do texto normal
<ul> Define uma lista não ordenada
<var> Define uma variável
<video> Define conteúdo de vídeo embutido
<wbr> Define uma possível quebra de linha
